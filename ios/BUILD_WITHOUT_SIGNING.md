# Сборка iOS без подписи кода

Проект настроен для сборки без подписи кода с использованием флага `--no-codesign`.

## Изменения в проекте

В файле `ios/Runner.xcodeproj/project.pbxproj` были добавлены следующие настройки для всех конфигураций (Debug, Release, Profile):

- `CODE_SIGN_IDENTITY = ""` - пустая идентичность подписи
- `CODE_SIGNING_REQUIRED = NO` - подпись не требуется
- `CODE_SIGNING_ALLOWED = NO` - подпись не разрешена
- `DEVELOPMENT_TEAM = ""` - пустая команда разработки
- `PROVISIONING_PROFILE_SPECIFIER = ""` - пустой профиль провижининга

## Команда для сборки

```bash
flutter build ios --release --no-codesign
```

## Примечания

- Приложение будет собрано, но не сможет быть установлено на устройство без ручной подписи
- Для установки на устройство потребуется:
  1. Открыть проект в Xcode: `open ios/Runner.xcworkspace`
  2. Выбрать Development Team в настройках подписи
  3. Подписать приложение вручную

## Альтернатива: сборка для симулятора

Если нужно только протестировать приложение, можно собрать для симулятора:

```bash
flutter build ios --simulator --release
```

Это не требует подписи кода.
